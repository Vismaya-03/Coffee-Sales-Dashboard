# Coffee-Sales-Dashboard
This project features an interactive Coffee Sales Dashboard built using excel. It provides a clear overview of sales performance across various products, store locations, time periods etc.

# Main Objective of the Project
The main objective is to analyze the reatil sales data to gain actionable insights that will enhance the performance of the coffee shop.

# Key Features
- Interactive slicers for dynamic filtering
- Visual KPIs showing total sales, quantity sold, best selling products
- Trend analyis for weekly and monthly sales
- Product wise breakdown for different product categories
- Clean and professional dashboard
  
#   Questions that were answered
- How do sales vary between days of the week and hours of the day
- What are the peak sales times
- Total sales revenue for each month.
- How sales varies across different store locations.
- What is the average price or order per person
- What are best selling products in terms of revenue

# What the project demonstrates
- Data Cleaning and Transformations
- Pivot Tables and Pivot Charts
- Measures
- Business insight extraction from raw data
- Dasboard creation
